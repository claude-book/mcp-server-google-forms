Instalar rápido

Os 3 passos essenciais

O caminho rápido para quem já tem experiência com o terminal. Precisa de contexto ou é a sua primeira vez? Use o Manual completo.

Pré-requisitos: Node.js 18+, Claude Code instalado e um client_secret*.json (OAuth do tipo “App para computador”) baixado do Google Cloud, com a Google Forms API ativada e o seu e-mail adicionado como usuário de teste na tela de permissão.

  1. Coloque o client_secret*.json na pasta de credenciais. Dica: rode o comando do passo 2 uma vez — ele cria a pasta ~/.config/mcp-server-google-forms/ sozinho e avisa que o arquivo falta. Aí é só soltar o JSON lá.
  2. Autorize com o Google:
    npx -p mcp-server-google-forms mcp-server-google-forms-token
    Aprove no navegador (em modo Testing, aceite o aviso de “app não verificado”).
  3. Registre no Claude Code:
    claude mcp add google-forms -- npx mcp-server-google-forms

Prefere clonar o repositório?

  1. git clone https://github.com/claude-book/mcp-server-google-forms.git && cd mcp-server-google-forms && npm install
  2. Ponha o client_secret*.json em credentials/ e rode npm run token.
  3. claude mcp add google-forms -- node "$(pwd)/src/server.js"

Depois é só pedir ao Claude, por exemplo: “crie um quiz de 5 perguntas sobre fotossíntese, 2 pontos cada, e me dê o link”.
