Código de exemplo do livro sobre Claude Code

Crie e edite Google Forms conversando com o Claude

mcp-server-google-forms é um servidor MCP local, em Node.js, que dá ao Claude Code ferramentas para criar formulários, adicionar perguntas, montar quizzes, publicar e ler respostas — tudo pela conversa.

Como funciona

O servidor roda no seu computador e conversa com a API oficial do Google Forms usando uma autorização que só você concede. Nenhum dado passa por servidores de terceiros: é a sua máquina falando diretamente com o Google.

Comece em quatro passos

  1. Clone o repositório e rode npm install.
  2. No Google Cloud, ative a Google Forms API e baixe credenciais OAuth do tipo “App para computador” para a pasta credentials/.
  3. Rode npm run token e autorize no navegador.
  4. Registre no Claude Code: claude mcp add google-forms -- node "$(pwd)/src/server.js"

O passo a passo completo, com solução de problemas, está no README.
